2023年12月和今年3月，集团公司党委会、总经理办公会、董事会先后听取公司安全生产工作和燃气安全管理情况汇报，从做好岁末年初安全工作、防范应对低温雨雪天气、深化城镇燃气安全整治等方面提出要求。按照会议精神，今年上半年重点组织了建筑施工、融雪防冻、防汛排渍、有限空间、消防安全等12个方面专项检查，督促能源集团开展“问题管网”专项治理，及时消除问题隐患，提升安全管理水平。根据会议安排，现将集团公司2024年上半年安全生产总体情况和下半年重点工作安排简要汇报如下：
一、上半年安全生产总体情况
2024年，集团公司围绕治本攻坚三年行动工作主线，聚焦“六项能力”提升，全面抓实安全生产、应急能力、管理体系固本强基，有效防范化解重大安全风险，重点完成以下工作：一是修订集团公司安全生产考核细则，逐级开展诚信承诺活动，优化安全管理“三项清单”，完善重点现场“一图、一牌和两单、两卡”。二是推进责任网络体系建设，完成244处重点部位风险评估分级，上半年公司领导分组带队督查3次，督办责任人开展现场检查225次，督促落实整改建议和工作要求295条。三是部署开展安全生产治本攻坚三年行动（2024-2026年），明确十大行动和22项具体任务，制定2024年工作清单，确保三年行动走深走实。四是严格落实集团公司党委会专题研究燃气安全工作的精神，下发安全督办单，组织督促能源集团开展“问题管网”整治，形成年度管网更新改造清单150项、共计145公里。五是围绕“人人讲安全、个个会应急—畅通生命通道”的安全月主题，组织各单位部署落实“四个一”，广泛开展隐患随手拍、专题学习、预案演练、宣传咨询、警示教育等系列活动。六是开展火灾集中治理“百日行动”，对所辖18处市级消防重点单位、59处人员密集场所、296处“九小场所”开展拉网式排查。七是修订集团公司生产安全事故和防汛排渍应急预案，完成应急资源普查摸底工作，安全应对春节前后2轮强降雪过程。八是组织做好今年防汛备汛工作，成立500人规模的市级防汛抢险城投中队，对194根穿江涉堤管线和900余处围墙围挡、工棚开展多轮次排查，发现隐患72处，已整改49处，剩余23处正在整改中。
二、下半年重点工作安排
今年上半年，能源集团一燃气管道施工项目发生一起高坠事故，说明我们部分单位在基层管理，特别是相关方的管理上，存在监管不够严格、责任落实不严的问题。此外，按照水文气象部门预测，预计今年主汛期长江流域降水总体偏多，可能出现较重汛情。针对面临的复杂形势和突出问题，围绕做好建国75周年等政治敏感期安全工作，按照集团公司“一个总体定位”“六项核心功能”“五大工作举措”的转型发展思路，下半年将重点落实以下工作：
（一）层层树牢安全发展理念。逐级开展《深入学习贯彻习近平关于应急管理的重要论述》专题研讨，提升统筹发展和安全能力。健全完善全员安全责任清单和工作清单，修订集团公司安全生产管理办法。深化安全生产领域不担当不作为突出问题整治。
（二）持续做好防汛排渍工作。层层夯实防汛抗旱责任体系，按照应对超标洪水、超标降雨的原则，督促各单位认真抓好以防雷防汛、防暑降温、防火防爆、防人员溺亡为重点的汛期安全工作。加强各级防汛队伍建设，定期开展实战实训。参与组建国家应急排涝基地应急队伍，完成市级防汛抢险任务。
（三）全力推动燃气隐患整治。督促能源集团深入开展“问题管网”整治攻坚，完成燃气管网风险评估工作，加快实施年度145公里燃气管道更新改造计划。持续加大安全用气宣传力度，加强用户端安全检查，严格执行入户安检制度。加快燃气管网物联感知设备等新技术建设与应用。
（四）全面强化基层安全管理。加强基层安全文化建设，推进安全管理标准化班组创建，建立“一对一”帮扶机制，力争年底新增2家以上全国安全管理标准化一级班组。开展相关方专项治理行动，强立推行“两同步、三严格”管理机制（同步管理、同步考核，严格资格准入、严格作业报备、严格日常检查）。
（五）推进治本攻坚三年行动。开展安全生产治本攻坚三年行动（2024年-2026年）专题宣贯，定期邀请第三方专家，开展现场指导帮扶，推动重大隐患排查整治。围绕十大行动和22项具体任务，对照2024年工作清单，高质量、高标准完成年度工作任务。
（六）强化安全生产科技支撑。聚焦安全防范、监测预警、事故处置，在供排水、城镇燃气、城市运维、项目建设、相关方管理等方面，推动安全信息化、数字化建设，提升本质安全运行水平。组织做好安全应急补助项目申报迎检工作，力争通过评审。
